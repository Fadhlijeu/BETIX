# BETIX

Betix adalah aplikasi pemesanan tiket bioskop berbasis web yang dirancang sebagai pengganti sistem tiket bioskop tradisional.

## Fitur Utama

- **Reservasi Tiket**: Pemesanan tiket dengan pilihan jam tayang dan kursi.
- **Manajemen Studio**: Administrasi studio bioskop including kursi, jam tayang, dan harga.
- **Reservasi**: Sistem reservasi dengan kursi dan total harga.

## Instalasi

1. Pastikan Node.js dan MySQL terinstall.
2. Clone repository ini.
3. Install dependensi:

```bash
npm install
```

4. Setup database sesuai dengan skema yang disediakan.
5. Jalankan server:

```bash
node server.js
```

## Teknologi yang Digunakan

- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Frontend**: Vanilla JavaScript (HTML, CSS, JS)
